Wordle Solver

In my free time, I created a program to solve the popular game Wordle. One of the functions of the program is to find possible answers to a daily Wordle puzzle but I also included another functionality. I used my knowledge of data structures and algorithms to run every possible Wordle game and find the best possible starting word (which I found to be the word "Soare"). You can access a github repository for the project here.

Working Y86-64 Processor

As part of the Computer Organization course at Texas A&M, I got the opportunity to construct a working processor using the Y86-64 instruction set in the program Logisim. The processor is built entirely out of logic gates and every component of the processor was created by me. You can access a github repository for the project here which includes three different codes (with their Y86-64 encodings) that can be executed by the processor, additional information about the design architecture of the processor, and the Logisim file for the processor itself.
